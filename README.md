# FM Design System

A professional UI component library for FeedMob — built with Web Components and Motion.

## Status

Work in Progress — Not ready for production use.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Usage

```html
<script type="module" src="./dist/fm-ds.js"></script>

<fm-button variant="primary">Get Started</fm-button>
<fm-card hoverable>
  <span slot="header">Card Title</span>
  Card content here
</fm-card>
<fm-badge variant="success" dot>Live</fm-badge>
```
