Design System

Professional UI components for FeedMob — built with Web Components & Motion

Getting Started

Work in Progress — This design system is currently under development and not ready for production use. 

# Install via npm
npm install @fm-agency/design-system

Then import in your JavaScript:

import '@fm-agency/design-system';
 
<!-- Add this to your HTML head -->
<script type="module" src="https://unpkg.com/@fm-agency/design-system/dist/fm-ds.js"></script>

Or download and use locally:

<script type="module" src="./dist/fm-ds.js"></script>
 
<!-- Just use the tags! -->
<fm-button variant="primary">Get Started</fm-button>

<fm-card hoverable>
  <span slot="header">Campaign Performance</span>
  Track real-time metrics across all active campaigns.
</fm-card>

<fm-badge variant="success" dot>Live</fm-badge>

That's it! FM Design System components work like native HTML elements.

Theming

Override CSS custom properties to customise the design system. All components inherit from these theme tokens. 

/* Override theme tokens */
:root {
  --fm-color-primary: #0D9488;
  --fm-color-primary-light: #14B8A6;
  --fm-color-primary-dark: #0F766E;
  --fm-color-secondary: #1E293B;
  --fm-font-family: 'Inter', system-ui, sans-serif;
  --fm-radius-md: 8px;
}

fm-button

Get Started Learn More View Case Study Read Docs
Small Medium Large
Active Disabled Disabled Outline

Attributes

NameTypeDefaultDescription
variant string "primary" Button style: primary, secondary, outline, ghost
size string "md" Button size: sm, md, lg
disabled boolean false Disables the button interaction

Animations

  • Entrance: Subtle fade-in + upward slide with spring
  • Hover: Refined lift with shadow enhancement
  • Press: Tactile press-down with spring physics
  • Focus: Teal ring indicator

<!-- Sizes -->

<!-- Disabled state -->
<fm-button variant="primary" disabled>Disabled</fm-button>

<!-- Listen for tab changes -->
<script>
  document.querySelector('fm-tabs').addEventListener('fm-tab-change', (e) => {
    console.log('Tab changed:', e.detail.index, e.detail.label);
  });
</script>

fm-alert

A new campaign template has been added to your library. Check it out in the templates section. Your campaign "Summer Sale 2026" has been successfully published and is now live. You have used 85% of your monthly budget. Consider adjusting your spend to avoid overages. Unable to sync with analytics provider. Please check your API credentials and try again. Simple alert without title or dismiss button — just informational text.

Attributes

NameTypeDefaultDescription
variant string "info" Alert style: info, success, warning, error
title string "" Alert title text
dismissible boolean false Shows close button when present

Slots

NameDescription
(default)Alert message content
iconCustom icon (overrides default)

Events

NameDescription
dismissFired when alert is dismissed

Animations

  • Entrance: Slide in from top with fade and spring physics
  • Exit: Slide up and fade out when dismissed
 
<!-- Info alert with title and dismiss button -->
<fm-alert variant="info" title="New Campaign Available" dismissible>
  A new campaign template has been added to your library.
</fm-alert>

<!-- Success alert -->
<fm-alert variant="success" title="Campaign Published" dismissible>
  Your campaign has been successfully published and is now live.
</fm-alert>

<!-- Warning alert -->
<fm-alert variant="warning" title="Budget Alert" dismissible>
  You have used 85% of your monthly budget.
</fm-alert>

<!-- Error alert -->
<fm-alert variant="error" title="Connection Failed" dismissible>
  Unable to sync with analytics provider.
</fm-alert>

<!-- Simple alert without title or dismiss -->
<fm-alert variant="info">
  Simple informational message without title.
</fm-alert>

<!-- Custom icon via slot -->
<fm-alert variant="info" title="Custom Icon" dismissible>
  <svg slot="icon" width="20" height="20" viewBox="0 0 20 20">...</svg>
  This alert uses a custom icon.
</fm-alert>

<!-- Listen for dismiss event -->
<script>
  document.querySelector('fm-alert').addEventListener('dismiss', () => {
